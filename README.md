## ⚙️ Installation Guide
### 🔧 Prerequisites
Java JDK 17+
Node.js & npm
MySQL / MariaDB
Maven

### 🔹 Steps
1️⃣ Clone the Repository
```
git clone https://github.com/your-username/betech-computers.git
cd betech-computers
```
2️⃣ Setup Backend (Spring Boot)
```
cd backend
mvn spring-boot:run
```
3️⃣ Configure Database

Update application.properties:
```
spring.datasource.url=jdbc:mysql://localhost:3306/betech_db
spring.datasource.username=root
spring.datasource.password=your_password
spring.jpa.hibernate.ddl-auto=update
```
4️⃣ Run Frontend
```
cd frontend
npm install
npm start
```
